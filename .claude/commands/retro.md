# /retro

Run a quick retrospective on the current or most recent AI-assisted session.

Review what happened and give structured feedback:

1. **What worked well?**
   - Which prompts produced good results on the first try?
   - Which patterns or tools were most effective?

2. **What didn't work?**
   - Where did the AI struggle, hallucinate, or go off track?
   - Which prompts needed multiple iterations?

3. **What should be saved?**
   - Any prompt worth saving as a slash command?
   - Any pattern worth encoding as a skill?
   - Any spec worth saving as a template?
   - Any `CLAUDE.md` rule that helped?

4. **What should be tuned?**
   - Skills that need refinement?
   - Commands that could be more specific?
   - `CLAUDE.md` rules to add or adjust?

5. **Action items**: 1-3 specific things to do before the next session.
   - e.g., "Save the dashboard spec as `templates/dashboard.md`."
   - e.g., "Add an error-handling principle to the API skill."

Keep it brief and actionable — not a lengthy report.
